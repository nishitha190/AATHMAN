# AATHMAN
# AATHMAN – AI-Powered Multi-Modal Mood Tracker

# Overview
# AATHMAN is an Artificial Intelligence based multi-modal emotion detection system designed to monitor and improve mental well-being. The system analyzes user emotions using multiple input modalities such as text, voice, and sign language, and provides meaningful insights through visualization and personalized feedback. It aims to help users understand their emotional patterns and improve their mental health through an intelligent and accessible platform.

# Objectives
# The main objectives of this project are:

# To detect emotions using text, speech, and gesture inputs
# To provide personalized mental health insights using AI techniques
# To develop a user-friendly and accessible system
# To track emotional patterns over time for better self-awareness

# Features
# The system includes text-based emotion detection using natural language processing, voice emotion analysis using speech-to-text models, and sign language recognition using computer vision techniques. It also provides an interactive dashboard where #users can visualize emotional trends. In addition, the application includes an AI-powered chatbot that offers personalized #suggestions and guidance based on the user’s emotional state.

# System Architecture
# The system works by collecting input from the user in the form of text, voice, or sign language gestures. The voice input is converted into text using a speech-to-text model, while gesture inputs are captured through a webcam and processed using feature extraction techniques. The processed data is analyzed using natural language processing and deep learning models to identify emotions. The outputs from all modalities are combined using a multimodal fusion approach to improve accuracy and reliability.

# Technologies Used
# The project is implemented using Python as the primary programming language. Streamlit is used for building the user interface. Machine learning libraries such as Scikit-learn are used for classification tasks. Natural language processing and large language models are used for text analysis, while Whisper is used for speech-to-text conversion. Computer vision techniques using OpenCV and MediaPipe are used for gesture recognition.

# Performance
# The system achieves approximately 89 to 90 percent accuracy for text-based and voice-based emotion detection. Sign language recognition achieves accuracy in the range of 85 to 88 percent. By combining all modalities, the overall system achieves an accuracy of approximately 94 percent, showing the effectiveness of the multi-modal approach.

# Applications
# This system can be used in various domains including:
# Mental health monitoring and emotional analysis
# Healthcare and counseling support systems
# Assistive technologies for users with communication difficulties
# Personal mood tracking and self-awareness tools

# Future Scope
# The system can be further improved by supporting multiple languages and enhancing sign language recognition capabilities. #Advanced models can be used to improve accuracy. Integration with wearable devices can enable real-time monitoring, and the #chatbot can be enhanced to provide more context-aware and personalized recommendations.

# Conclusion
AATHMAN is a comprehensive multi-modal emotion detection system that combines text, voice, and gesture inputs to provide accurate and meaningful insights into mental well-being. The system not only improves emotion detection accuracy but also helps users understand and manage their emotional patterns effectively. It serves as a valuable tool for promoting mental health and self-awareness.
